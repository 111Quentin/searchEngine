意媒曝囧叔留队尤文不容乐观 斑马军团已着手联系德尚

腾讯体育4月30日讯 阿莱格里多次在公开场合表示下赛季会继续留在尤文图斯执教，但是意大利媒体《米兰体育报》却认为囧叔是否会留在尤文图斯取决于他与尤文图斯主席阿涅利的会面，这次会议会是决定性的。一旦囧叔离队，他不会缺少未来下家，英超球队、国米以及巴黎都非常欢饮这位意大利教练，而斑马军也有备胎，德尚这位法国主帅得到了帕拉蒂奇以及内德维德的欣赏。

囧叔今夏要离开尤文？

阿莱格里在过去的五个赛季带队尤文图斯获得了五次意甲冠军，同时五个赛季带队杀入两次欧冠决赛。在本赛季的欧冠联赛1/4淘汰赛输给阿贾克斯后，阿莱格里表示自己会继续留在尤文图斯，而在公开场合，阿莱格里多次表达这一论点。

意大利媒体《米兰体育报》并没有这么乐观，这家媒体称，阿莱格里是否会继续留在尤文，与主席阿涅利的会议是决定性的，两者需要在球队未来规划以及球员转会达成一致的情况下才可以继续合作。

不仅如此，阿莱格里目前也受到了众多英超球队、国米以及巴黎的邀请。尤其是巴黎，根据《都灵体育报》的消息，在过去的几个小时巴黎的高层正在认真的考虑将阿莱格里从尤文图斯挖走。

因此在以上的背景下，尤文图斯高层也有了替换阿莱格里的人选。孔蒂是个不错的选择，但是《米兰体育报》称，这位昔日尤文教练回归难度不小，克洛普？这个也是遥远的选择。而这家媒体称，目前在法国执教的德尚会是尤文图斯的主要目标，尤文图斯的体育总监帕拉蒂奇以及副主席内德维德的都对德尚十分感兴趣。

德尚带队法国赢得了2016年欧洲杯的亚军，同时在2018年世界杯赢得了世界杯的冠军，他的执教能得到了认可。不仅如此，德尚还曾经在尤文踢过球，同时也曾经担任过尤文图斯的教练，这种背景也加大了他到尤文再次执教的可能。
